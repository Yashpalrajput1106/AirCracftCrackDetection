# AirCracftCrackDetection
This project is a deep learning-based image classification system developed as part of an engineering mini project. It detects whether a surface of an aircraft is cracked or uncracked using a Convolutional Neural Network (CNN).  The model is trained on image data and evaluates performance using metrics like confusion matrix and ROC curve.

## Features
Image classification (Cracked vs Uncracked)
CNN-based deep learning model
Data preprocessing and normalization
Train-test split for evaluation
Confusion matrix visualization
ROC curve and AUC score analysis
Prediction visualization on test images

## Technologies Used
Python
TensorFlow / Keras
OpenCV
NumPy
Matplotlib
Seaborn
Scikit-learn

## Evaluation Metrics
Accuracy
Confusion Matrix (Train & Test)
ROC Curve
AUC Score

## Output
Heatmaps of confusion matrices
ROC curve visualization
Sample predictions on test images
